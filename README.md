# inkfilr

inkfilr is a React-based landing page revamp of the original portfolio template, rebuilt to keep the same visual structure and core color identity while modernizing interaction and maintainability.

## What changed

- Rebuilt from static HTML/jQuery into React + Vite
- Styled with Tailwind CSS and brand tokens
- Replaced icon usage with Lucide icons
- Kept the same section flow and image-driven layout (Hero, About, Services, Projects, Contact)
- Added smooth scrolling, sticky navigation, and accessible focus/motion behavior

## Tech stack

- React
- Vite
- Tailwind CSS
- Lucide React
- ESLint

## Getting started

1. Install dependencies:

   npm install

2. Start development server:

   npm run dev

3. Build for production:

   npm run build

4. Preview production build:

   npm run preview

5. Run lint checks:

   npm run lint

## Notes

- All project images are reused from the existing repository assets.
- This repository currently focuses on a single-page landing experience.
